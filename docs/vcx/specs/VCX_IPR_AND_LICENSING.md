# VCX IPR and Licensing Posture (v1)

**Status**: draft
**Owner**: Core Runtime + Legal
**Last reviewed**: 2026-02-20

## Purpose

Record the intended legal/IPR posture for VCX standard publication.

## Intended Direction

- Specification text: permissive documentation license.
- Reference implementation: permissive OSS license.
- Conformance vectors: redistributable under permissive terms.

## Patent/Claims Posture

- Contributors are expected to disclose known patent claims relevant to submitted changes.
- Potentially encumbered contributions require explicit legal review before acceptance.

## Required Artifacts Before S2 Exit

1. Final chosen licenses for specs and code.
2. Contributor patent disclosure statement.
3. Public IPR FAQ with policy contacts.

## Current Status

This document is a policy scaffold and requires legal sign-off before being marked `active`.
